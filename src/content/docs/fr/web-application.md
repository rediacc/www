# Guide d'utilisation de la plateforme Rediacc

> **Note de traduction**: Ce document est actuellement disponible uniquement en [anglais](/en/web-application) et en [turc](/tr/web-application). Une traduction française est en cours.

## Présentation

**Rediacc** est une plateforme cloud offrant des services de sauvegarde assistés par IA.

Ce guide décrit l'utilisation de base de l'interface web sur [https://www.rediacc.com/](https://www.rediacc.com/).

### Objectif de ce guide

- Aider les nouveaux utilisateurs à s'adapter rapidement à la plateforme
- Expliquer les fonctions de base (gestion des ressources, sauvegarde) étape par étape
