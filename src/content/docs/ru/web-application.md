# Руководство пользователя платформы Rediacc

> **Примечание о переводе**: Этот документ в настоящее время доступен только на [английском](/en/web-application) и [турецком](/tr/web-application) языках. Русский перевод находится в разработке.

## Обзор

**Rediacc** — это облачная платформа, предлагающая услуги резервного копирования с поддержкой ИИ.

Это руководство описывает основное использование веб-интерфейса на [https://www.rediacc.com/](https://www.rediacc.com/).

### Цель этого руководства

- Помочь новым пользователям быстро адаптироваться к платформе
- Объяснить основные функции (управление ресурсами, резервное копирование) пошагово
