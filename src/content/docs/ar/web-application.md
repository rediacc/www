# دليل مستخدم منصة Rediacc

> **ملاحظة الترجمة**: هذا المستند متاح حاليًا فقط [بالإنجليزية](/en/web-application) و[بالتركية](/tr/web-application). الترجمة العربية قيد الإعداد.

## نظرة عامة

**Rediacc** هي منصة سحابية تقدم خدمات النسخ الاحتياطي بدعم الذكاء الاصطناعي.

يصف هذا الدليل الاستخدام الأساسي لواجهة الويب على [https://www.rediacc.com/](https://www.rediacc.com/).

### الغرض من هذا الدليل

- مساعدة المستخدمين الجدد على التكيف بسرعة مع المنصة
- شرح الوظائف الأساسية (إدارة الموارد، النسخ الاحتياطي) خطوة بخطوة
